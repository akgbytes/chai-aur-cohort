# Problem Statement 6

You need to determine what action to take based on traffic light colors:

- "Red" → Stop
- "Yellow" → Slow Down
- "Green" → Go
- “Blue” → Invalid Color

## Challenge

Write a function that uses switch-case to return the correct action.

#### ✅ Input:

```json
{ "color": "Red" }
```

#### 🎯 Expected Output:

```json
"Stop"
```
